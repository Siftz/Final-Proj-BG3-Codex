"# Final-Project"
 Hey welcome to my codex for Baldurs Gate 3. I wanted to design a page that could list all of the useful wearable gear in the game into one place. 
 The main usage of this site is to be able to look at a vast database of over 300 items and be able to see where those items are in the game without revealing to much spoilers for the game as a whole.
 Initially planned on having a map tied to the items but decided this would pose a breach of spoiling the game for new players. I added a mannequin so that users could add and remove them to see what they want the most.
 I added build guides for the game as well built from other experts and my own homebrews that I enjoyed playing and hoped that other's would as well.
 My hope is that it will be a fun place to see where the gear is and the descriptions of the gear as well throughout the game. 
 This was a massive undertaking and the database was the main source of my initial time and developement and im glad i was able to pracice with MySQL because of it.
